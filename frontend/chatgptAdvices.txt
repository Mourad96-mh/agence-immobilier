1️⃣ Serve sitemap.xml correctly

Right now, your server serves the React build with:

app.use(express.static(distPath));
app.get('*', (req, res) => res.sendFile(path.join(distPath, 'index.html')));

⚠️ Problem: Requests like /sitemap.xml get caught by the wildcard * and return index.html, not XML. That’s why Google sometimes fails to read it.

Fix: Add an explicit route for sitemap.xml before the wildcard route:

// Serve sitemap.xml explicitly
app.get('/sitemap.xml', (req, res) => {
  res.sendFile(path.join(distPath, 'sitemap.xml'));
});

Now /sitemap.xml will always return the correct XML, even in production.

2️⃣ Serve robots.txt explicitly (optional but recommended)

You already redirect robots.txt for onrender.com, but you should also serve it for your real domain:

app.get('/robots.txt', (req, res) => {
  res.type('text/plain');
  res.send(
    `User-agent: *
Allow: /

Sitemap: https://www.mecalus.org/sitemap.xml`
  );
});

This ensures Googlebot always finds it.
